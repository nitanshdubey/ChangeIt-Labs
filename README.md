## ✨ Features

🎨 **Premium Design**
- Luxury aesthetic inspired by Stripe, Linear, Framer
- Generous whitespace and typography hierarchy
- Glass morphism effects and smooth gradients
- Modern card designs with soft shadows

⚡ **Performance Optimized**
- Lightning-fast load times
- SEO best practices
- Mobile-first responsive design
- Lazy loading and code splitting

🎯 **Conversion Focused**
- WhatsApp CTA on every important section
- Lead generation through contact forms
- Trust-building elements (testimonials, stats, founder story)

- **React 18** - UI library
- **Vite** - Build tool
- **TypeScript** - Type safety

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/nitanshdubey/ChangeIt-Labs.git
cd ChangeIt-Labs

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
